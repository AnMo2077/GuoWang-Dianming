# 随机点名与运算系统

一个现代化、功能丰富的随机点名应用，结合了随机运算功能，提供了直观的用户界面和多种实用特性。

## 主要功能

### 1. 随机点名与运算
- 随机抽取1-54之间的数字作为初始值
- 自动进行多步骤随机运算（+、-、×、÷）
- 确保运算结果始终在有效范围内
- 支持并行抽取多个结果（最多10个）

### 2. 可配置设置
- 灵活调整点名数量（1-10个）
- 可设置运算步骤数（1-5步）
- 实时显示当前抽取数量

### 3. 直观的可视化界面
- 动态数字滚动动画效果
- 步骤化运算过程展示
- 实时进度条显示整体进度
- 清晰的状态信息反馈

### 4. 历史记录管理
- 自动保存最近10次点名结果
- 记录完整的运算过程
- 支持一键清空历史记录

### 5. 主题切换
- 默认深色主题设计
- 支持切换浅色主题
- 主题偏好自动保存

### 6. 独特的视觉设计
- 炫彩渐变标题效果
- 毛玻璃边框设计
- 响应式布局适配各种屏幕
- 精心设计的动画过渡效果

## 技术特点

- **前端技术栈**：HTML5 + CSS3 + JavaScript (ES6+)
- **现代CSS特性**：渐变、毛玻璃效果、动画、过渡
- **响应式设计**：适配桌面和移动设备
- **本地存储**：使用localStorage保存用户偏好
- **模块化代码**：清晰的功能划分和代码结构
- **性能优化**：动画优化、减少DOM操作

## 使用方法

1. **配置设置**：
   - 在顶部配置面板设置需要抽取的数量
   - 选择运算步骤数

2. **开始点名**：
   - 点击"开始点名"按钮
   - 观察数字滚动和运算过程

3. **查看结果**：
   - 结果将显示在每个点名卡片中
   - 完整运算步骤会实时展示

4. **历史记录**：
   - 所有结果自动保存到历史记录面板
   - 点击"清空历史"可删除所有记录

5. **主题切换**：
   - 使用右上角的主题切换按钮切换深色/浅色主题

## 界面预览

- **动态视觉效果**：流畅的数字滚动和运算动画
